# Java Learning Journey

Welcome to my Java learning journey! This repository contains Java programming questions and solutions that I have worked on.

## Note

If you use or copy any of the code from this repository, please create a pull request with the screenshot of the output of the solution. This helps maintain the integrity of the solutions and ensures that everything works as expected.

## Structure

Each solution is stored in a folder.

The following are some of the key concepts I aim to cover as I progress:

- Basics of Java syntax
- Object-Oriented Programming (OOP)
- Data structures and algorithms (arrays, linked lists, trees, sorting algorithms, etc.)
- Exception handling
- Java libraries and APIs (Collections, Streams, etc.)
- Multithreading and concurrency
- Design patterns and principles

We will be following this guide : https://docs.oracle.com/javase/tutorial/ ( DOCUMENTATION )
